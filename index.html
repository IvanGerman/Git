Create local then send to Git (git init)

git init - создать Git репозиторий в папке на локальном ПК
git add index.html OR git add * OR git add . - ставит файл на отслежку Гитом
git rm --cached index.html OR git restore --staged index.html - убирает обратно с отслежки
git restore index.html - убирает последние изменения в файле index.html
git commit -m "first commit" - делаем commit/фиксацию текущего состояния файлов, их 'слепок'
gitK --all& - открывает GUI
git branch test1 - создаёт ещё одну ветку test1
git branch -D test1 - стирает ветку test1
git checkout test1 - попадаем в ветку test1
git checkout -b test2 - создаём и попадаем в ветку test2
